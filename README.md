# Dev Portfolio

A config driven portfolio to display your projects and accomplishments.

Fork it, update the constants in `src/data` folder and make it yours.

Live demo <a href="https://mrprince88.vercel.app" >here</a>.


Tech stack used:
- Next.js
- Tailwind CSS
- shadcn
- Lottie React
